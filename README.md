# sistema-asistencia
Sistema web de control de asistencia desarrollado con PHP y MySQL. Permite gestionar empleados y registrar entradas y salidas. Implementa procedimientos almacenados, triggers y transacciones para asegurar la integridad y automatización de los datos.
# Sistema de Control de Asistencia

Sistema web desarrollado con PHP y MySQL que permite gestionar empleados y registrar entradas y salidas de manera eficiente.

## Funcionalidades

- Registro, consulta y eliminación de empleados  
- Control de asistencia (entrada y salida)  
- Validación de empleados por cédula y departamento  
- Registro automático de acciones en historial  
- Uso de procedimientos almacenados  
- Implementación de triggers  
- Manejo de transacciones para asegurar la integridad de los datos  

## Tecnologías utilizadas

- HTML  
- CSS  
- PHP  
- MySQL  

## Instalación

1. Instalar XAMPP  
2. Copiar el proyecto en:
   C:\xampp\htdocs\
3. Importar la base de datos (asistencia.sql) en phpMyAdmin  
4. Ejecutar en el navegador:
   http://localhost/asistencia_pro

## Base de datos

El archivo de la base de datos se encuentra incluido en el proyecto como:
asistencia.sql

## Evidencia

El sistema fue probado en entorno local, mostrando el correcto funcionamiento del registro de asistencias, gestión de empleados y generación automática de historial.
